A high-performance, open-source, webapp for visualizing data from the STRING database.

Bridge the gap between raw interactome data and publication-quality visual insights.

Open StringScape View Source Code
High-Performance

Interactive Biological Networks

View and explore the full STRING network from any species. The app runs compute-heavy tasks on your local GPU, making it super fast and responsive.

Data Intelligence

Synchronized Views & Variables

Colour and size nodes based on any STRING or custom variables. Visualize these variables as histograms, pie charts, scatter plots, mind maps, or UMAP plots. All views are synchronized allowing you to select proteins in one view and see those same proteins selected in another view.

Dimensionality Reduction

Visualize High-Dimensional Embeddings

View 2D and 3D UMAP plots of both network- and sequence-embedding spaces. Easily identify functional and structural similarities among your proteins.

Visualize structures

Complex Structures Dashboard

View protein complex structures in an embedded Mol* viewer, allowing you to see how the proteins interact in 3D space.

Seamless workflow integration

Useful Links and Export Options

Export publication-quality images of any view alongside organized spreadsheets of data subsets. Other export options include FASTA and JSON (for submission to the AlphaFold Server). The app also has links to useful information on other websites such as STRING, IntAct, UniProt, NCBI, the Protein Data Bank, and relevant PubMed papers.

In-Browser Compute

Secure Sandbox Python Console

Run advanced analysis natively utilizing an integrated Python/pyodide console. These scripts can also be used to perform actions in the app via the same tool calls the AI agent uses.

Privacy First AI

Integrated AI Research Agent

Connect local LLMs via LM Studio for private, free, unlimited AI compute. The agent can answer questions about the data, create interactive guides, perform actions in the app on your behalf, read the apps own code, and help debug issues.
